[Section: সেট প্রকাশের পদ্ধতি] [Translation: Methods of Representing Sets]

[Section: তালিকা পদ্ধতি] [Translation: Roster Method]
{এ পদ্ধতিতে সেটের সকল উপাদান সুনির্দিষ্টভাবে উল্লেখ করে দ্বিতীয় বন্ধনী {} এর মধ্যে আবদ্ধ করা হয় এবং একাধিক উপাদান থাকলে 'কমা' ব্যবহার করে উপাদানগুলোকে আলাদা করা হয়।} [Translation: In this method, all elements of a set are explicitly listed within curly brackets {} and separated by commas if there are multiple elements.] {যেমন, A = {a,b}, B = {2,4,6}, C = {নিলয়, তিশা, শুভ্রা} ইত্যাদি।} [Translation: For example, A = {a,b}, B = {2,4,6}, C = {Niloy, Tisha, Shubhra} etc.]

[Section: সেট গঠন পদ্ধতি] [Translation: Set Builder Method]
{এ পদ্ধতিতে সেটের সকল উপাদান সুনির্দিষ্টভাবে উল্লেখ না করে উপাদান নির্ধারণের জন্য সাধারণ ধর্মের উল্লেখ থাকে।} [Translation: In this method, instead of listing all elements, a common property of the elements is defined.] {যেমন: A = {x: x স্বাভাবিক বিজোড় সংখ্যা }, B = {x : x নবম শ্রেণির প্রথম পাঁচজন শিক্ষার্থী} ইত্যাদি।} [Translation: For example: A = {x: x is a natural odd number}, B = {x: x is the first five students of class nine} etc.] {এখানে, ':' দ্বারা 'এরূপ যেন' বা সংক্ষেপে 'যেন' (such that) বোঝায়।} [Translation: Here, ':' means 'such that'.] {যেহেতু এ পদ্ধতিতে সেটের উপাদান নির্ধারণের জন্য শর্ত বা নিয়ম (Rule) দেওয়া থাকে, এ জন্য এ পদ্ধতিকে Rule Method ও বলা হয়।} [Translation: Since this method defines the elements based on condition or rule, it is also known as the Rule method.]

[Example: Example 1]
{উদাহরণ ১.  A = {7,14, 21, 28} সেটটিকে সেট গঠন পদ্ধতিতে প্রকাশ কর।} [Translation: Example 1. Express the set A = {7, 14, 21, 28} using the Set Builder Method.]
{সমাধান: A সেটের উপাদানসমূহ 7, 14, 21, 28।} [Translation: Solution: The elements of set A are 7, 14, 21, 28.]
{এখানে, প্রত্যেকটি উপাদান 7 দ্বারা বিভাজ্য, অর্থাৎ 7 এর গুণিতক এবং 28 এর বড় নয়।} [Translation: Here, each element is divisible by 7, i.e., a multiple of 7 and not greater than 28.]
$... A = \{x : x, 7 \text{ এর গুণিতক এবং } 0 < x \le 28\}$ [Translation: ... A = {x: x is a multiple of 7 and 0 < x ≤ 28}]

[Example: Example 2]
{উদাহরণ ২. B = {x: x, 28 এর গুণনীয়ক} সেটটিকে তালিকা পদ্ধতিতে প্রকাশ কর।} [Translation: Example 2. Express the set B = {x: x is a factor of 28} using the roster method.]
{সমাধান: এখানে, $28 = 1 \times 28 = 2 \times 14 = 4 \times 7$} [Translation: Solution: Here, $28 = 1 \times 28 = 2 \times 14 = 4 \times 7$]
{... 28 এর গুণনীয়কসমূহ 1, 2, 4, 7, 14, 28} [Translation: ... the factors of 28 are 1, 2, 4, 7, 14, 28]
{নির্ণেয় সেট B = {1, 2, 4, 7, 14, 28}} [Translation: Therefore the set B is {1, 2, 4, 7, 14, 28}]

[Section: সসীম সেট (Finite Set)] [Translation: Finite Set]
[Definition: Finite Set] যে সেটের উপাদান সংখ্যা গণনা করে নির্ধারণ করা যায়, তাকে সসীম সেট বলে। [Translation: A set whose elements can be counted is called a finite set.]
যেমন, $D = \{x,y,z\}$, $E = \{3,6,9,...,60\}$, $F = \{x: x$ মৌলিক সংখ্যা এবং $30 < x < 70\}$ ইত্যাদি সসীম সেট। [Translation: For example, D={x,y,z}, E={3,6,9,...,60}, F={x: x is a prime number and 30 < x < 70} etc. are finite sets.]
এখানে, D সেটে 3 টি, E সেটে 20 টি এবং F সেটে 9 টি উপাদান আছে। [Translation: Here, D has 3 elements, E has 20 elements, and F has 9 elements.]

[Section: অসীম সেট (Infinite Set)] [Translation: Infinite Set]
[Definition: Infinite Set] যে সেটের উপাদান সংখ্যা গণনা করে শেষ করা যায় না, তাকে অসীম সেট বলে। [Translation: A set whose elements cannot be counted is called an infinite set.]
যেমন, $A = \{x: x$ বিজোড় স্বাভাবিক সংখ্যা$\}$, স্বাভাবিক সংখ্যার সেট $N = \{1,2,3,4,...\}$, পূর্ণসংখ্যার সেট $Z = \{..., -3, -2, -1, 0, 1, 2, 3...\}$, মূলদ সংখ্যার সেট $Q = \{\frac{a}{b}: a$ ও $b$ পূর্ণসংখ্যা এবং $b \neq 0\}$, বাস্তব সংখ্যার সেট $R$ ইত্যাদি অসীম সেট। [Translation: For example, A = {x: x is an odd natural number}, the set of natural numbers N = {1, 2, 3, 4,...}, the set of integers Z = {..., -3, -2, -1, 0, 1, 2, 3,...}, the set of rational numbers Q = {a/b : a and b are integers and b ≠ 0}, the set of real numbers R, etc. are infinite sets.]

[Example: Example 4] দেখাও যে, সকল স্বাভাবিক সংখ্যার সেট একটি অসীম সেট। [Translation: Show that the set of all natural numbers is an infinite set.]
সমাধান: স্বাভাবিক সংখ্যার সেট $N = \{1, 2, 3, 4, 5, 6, 7, 8, ...\}$ [Translation: Solution: The set of natural numbers is N = {1, 2, 3, 4, 5, 6, 7, 8, ...}]
$N$ সেট থেকে বিজোড় স্বাভাবিক সংখ্যাসমূহ নিয়ে গঠিত সেট $A = \{1, 3, 5, 7, ...\}$ [Translation: The set of odd natural numbers from N is A = {1, 3, 5, 7, ...}]
$N$ সেট থেকে জোড় স্বাভাবিক সংখ্যাসমূহ নিয়ে গঠিত সেট $B = \{2, 4, 6, 8, ...\}$ [Translation: The set of even natural numbers from N is B = {2, 4, 6, 8, ...}]
$N$ সেট থেকে 3 এর গুণিতকসমূহের সেট $C = \{3, 6, 9, 12, ...\}$ ইত্যাদি। [Translation: The set of multiples of 3 from N is C = {3, 6, 9, 12, ...} etc.]
এখানে, $N$ সেট থেকে গঠিত উপরের সেটসমূহের উপাদান সংখ্যা গণনা করে নির্ধারণ করা যায় না। [Translation: Here, the number of elements in the above sets formed from set N cannot be determined by counting.]
ফলে $A, B, C$ অসীম সেট। [Translation: Therefore A, B, C are infinite sets.]
... $N$ একটি অসীম সেট। [Translation: Therefore N is an infinite set]

[Section: Definition of Relation]
যদি A ও B দুইটি সেট হয় তবে সেটদ্বয়ের কার্তেসীয় গুণজ A × B সেটের অন্তর্গত ক্রমজোড়গুলোর [Translation: If A and B are two sets, then the ordered pairs within the Cartesian product A × B of those sets]
অশূন্য উপসেট R কে A সেট হতে B সেটের একটি অন্বয় বা সম্পর্ক বলা হয়। [Translation: a non-empty subset R is called a relation from set A to set B]
এখানে, R সেট A × B সেটের একটি উপসেট অর্থাৎ, $R \subseteq A \times B$ [Translation: Here, set R is a subset of A × B i.e.,  $R \subseteq A \times B$]

[Example: 15]
মনে করি, A = {3,5} এবং B = {2,4} [Translation: Let A = {3,5} and B = {2,4}]
$\therefore A \times B = \{3,5\} \times \{2,4\} = \{(3, 2), (3, 4), (5, 2), (5, 4)\}$
$\therefore R \subseteq \{(3,2), (3, 4), (5, 2), (5, 4)\}$

যখন A সেটের একটি উপাদান x ও B সেটের একটি উপাদান y এবং (x, y) ∈ R হয় তবে লেখা [Translation: When x is an element of set A and y is an element of set B, and (x,y) ∈ R, it's written]
হয় xRy এবং পড়া হয় x, y এর সাথে অন্বিত (x is related to y) অর্থাৎ উপাদান x, উপাদান [Translation: as xRy, and read as 'x is related to y' or that element x is related to element]
y এর সাথে R সম্পর্কযুক্ত। [Translation: y with respect to R]
যদি x > y শর্ত হয় তবে, R = {(3,2), (5, 2), (5, 4)} [Translation: If the condition is x > y, then R = {(3,2), (5, 2), (5, 4)}]
এবং যদি x < y শর্ত হয় তবে, R = {(3,4)} [Translation: And if the condition is x < y, then R = {(3,4)}]
আবার, A সেট হতে A সেটের একটি অন্বয় অর্থাৎ $R \subseteq A \times A$ হলে, R কে A এর অন্বয় বলা হয়। [Translation: Again, a relation from set A to itself (i.e.,  $R \subseteq A \times A$), is called a relation on A]

A এবং B দুইটি সেটের উপাদানগুলোর মধ্যে সম্পর্ক দেওয়া থাকলে x ∈ A এর সংগে সম্পর্কিত [Translation: Given a relationship between the elements of two sets A and B, with x belonging to A and related to]
y ∈ B নিয়ে যে সব ক্রমজোড় (x, y) পাওয়া যায়, এদের অশূন্য উপসেট হচ্ছে একটি অন্বয়। [Translation: all the ordered pairs (x,y) where y belongs to B, these non-empty subsets are called a relation]

[Example: 16]
যদি P = {2, 3, 4}, Q = {4,6} এবং P ও Q এর উপাদানগুলোর মধ্যে y = 2x [Translation: If P = {2, 3, 4}, Q = {4,6} and between the elements of P and Q, the relation is y = 2x]

প্রশ্নানুসারে, R = {(x, y) : x ∈ P, y ∈ Q এবং y = 2x} [Translation: According to the problem, R = {(x, y) : x ∈ P, y ∈ Q and y = 2x}]
এখানে, $P \times Q = \{2, 3, 4\} \times \{4,6\} = \{(2, 4), (2, 6), (3, 4), (3, 6), (4, 4), (4, 6)\}$
$\therefore R = \{(2, 4), (3, 6)\}$
নির্ণেয় অন্বয় {(2, 4), (3, 6)} [Translation: The required relation is {(2, 4), (3, 6)}]

[Definition: $f(y)$] $f(y) = \frac{y^3 - 3y^2 + 1}{y(1-y)}$
{সমাধান: $f(y) = \frac{y^3 - 3y^2 + 1}{y(1-y)}$} [Translation: Solution:]

{$\therefore f(\frac{1}{y}) = \frac{(\frac{1}{y})^3 - 3(\frac{1}{y})^2 + 1}{\frac{1}{y}(1-\frac{1}{y})} = \frac{\frac{1}{y^3} - \frac{3}{y^2} + 1}{\frac{1}{y}(\frac{y-1}{y})} = \frac{\frac{1-3y+y^3}{y^3}}{\frac{y-1}{y^2}} = \frac{1-3y+y^3}{y^3} \times \frac{y^2}{y-1} = \frac{1-3y+y^3}{y(y-1)}$}

B = {x : x জোড় সংখ্যা এবং [even number and] x ≤ 6} = {2,4,6}
.: A ∪ B = {2,3,5} ∪ {2, 4, 6} = {2,3,4,5,6} ... (1)
A \ B = {2,3,5} - {2,4,6} = {3,5}
B \ A = {2,4,6} - {2,3,5} = {4,6}
A ∩ B = {2,3,5} ∩ {2,4,6} = {2}
.: (A \ B) ∪ (B \ A) ∪ (A ∩ B) = {3, 5} ∪ {4, 6} ∪ {2} = {2, 3, 4, 5, 6} ... (2)
সুতরাং [Therefore] (1) ও (2) তুলনা করে পাই, [by comparing (1) and (2)]
A ∪ B = (A \ B) ∪ (B \ A) ∪ (A ∩ B)

[Section: গ] (2) হতে পাই, [From (2) we get]
C = A \ B = {3,5}
A ∩ C = {2,3,5} ∩ {3,5} = {3,5}
.: (A ∩ C) × B = {3,5} × {2, 4, 6}
= {(3, 2), (3, 4), (3, 6), (5, 2), (5, 4), (5, 6)} ......(3)
A × B = {2,3,5} × {2,4,6}
= {(2, 2), (2, 4), (2, 6), (3, 2), (3, 4), (3, 6), (5, 2), (5, 4), (5, 6)}
C × B = {3,5} × {2,4,6}
= {(3, 2), (3, 4), (3, 6), (5, 2), (5, 4), (5, 6)}
therefore (A × B) ∩ (C × B)
So (A ∩ C) × B = (A × B) ∩ (C × B)
